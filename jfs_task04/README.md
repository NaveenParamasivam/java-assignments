# Java Full Stack – Task 04

## Topic
Exception Handling & Collections


## Problems Implemented

### Q1 – Student Management System
- Uses parameterized constructor
- Validates age and name
- Custom checked exceptions implemented
- Package: `studentmanagementsystem`

### Q2 – Voter Eligibility Check
- Parameterized constructor
- Throws exception if age is less than 18
- Custom unchecked exception implemented
- Package: `electionsystem`

### Q3 – Weekday Finder
- Stores weekdays in an array starting from Sunday
- Accepts day index from user
- Handles array index out of bounds exception
- Package: `dayfinder`

### Q4 – Student Grades using HashMap
- Stores student names and grades
- Methods to add, remove, and display grades
- Uses HashMap collection
- Package: `studentgrades`

### Q5 – Stack using Collection Classes
- Stores integers using Stack
- Methods for push, pop, and empty check
- Uses Java Collection Framework
- Package: `collectionproblem`

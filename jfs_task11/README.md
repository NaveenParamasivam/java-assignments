# JFS Task 11: Spring Boot & Spring MVC Applications


## Overview
This repository contains the solutions for Task 11 of the Java Full-Stack curriculum. It consists of two separate Spring Boot web applications that demonstrate the use of Spring MVC, Spring Data JPA, Thymeleaf, and MySQL for building interactive, database-driven web applications.

## Projects

### 1. Question 1: Product Catalog System
A web application designed to manage a product catalog. Users can add new products via a form and view the entire catalog in a tabular format.

- **Key Features:** Add Product, Display Products, Filter by Category.
- **Tech Stack:** Spring Boot, Spring MVC, Spring Data JPA, MySQL, Thymeleaf.

### 2. Question 2: Feedback Application
A web application that allows users to submit feedback for books. The data is persisted to a MySQL database, and users receive a success confirmation upon submission.

- **Key Features:** Submit Feedback Form, View All Feedback.
- **Tech Stack:** Spring Boot, Spring MVC, Spring Data JPA, MySQL, Thymeleaf.

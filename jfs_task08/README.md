# DOM Pagination Task

## Description
This project demonstrates pagination using DOM manipulation with JavaScript.
The data is loaded dynamically and displayed in a table with pagination controls.

## Features
- Dynamic table data rendering
- First, Previous, Numbered, Next and Last pagination buttons
- Responsive design using Bootstrap and CSS
- No deprecated HTML tags used

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap

## How to Run
1. Open `index.html` in a browser
2. Use pagination buttons to navigate data

## Test Suite
The Zen Class test suite is included inside the HTML file as required.
